INMOSCRAPER DEMO
================

Abre este archivo:
- Inmoscraper.exe

Si Windows muestra una advertencia:
- pulsa "Más información"
- luego "Ejecutar de todas formas"

Cómo cerrar la demo:
- cierra la pestaña del navegador
- y después cierra la ventana de Inmoscraper

Si falla:
- usa el fallback run_inmoscraper.bat
- o contacta con la persona que te entregó esta demo
